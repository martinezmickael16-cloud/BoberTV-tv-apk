# 🦫 BoberTV APK Builder

Ce repo compile automatiquement l'APK BoberTV via GitHub Actions.

## Comment ça marche

1. Tu pousses du code sur la branche `main`
2. GitHub Actions compile automatiquement l'APK Android
3. L'APK est disponible dans **Releases** ou **Actions > Artifacts**

## Installation de l'APK

1. Télécharge `app-debug.apk` depuis les Releases
2. Sur Android → **Paramètres** → **Sécurité** → **Sources inconnues** → Activer
3. Ouvre le fichier APK → **Installer**
4. Lance **BoberTV** depuis ton écran d'accueil ✅

## Avantages de l'APK

- User-Agent Android natif → accepté par les fournisseurs IPTV
- Icône sur l'écran d'accueil
- Plein écran natif
- Notifications possibles
